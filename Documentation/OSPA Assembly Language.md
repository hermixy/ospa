# OSPA Assembly Language

## Terminology

### Address

Format: @(Type)(Number)
Examples: @D1, @W5, @WX70

Address Type | Description
------------ | -----------
D            | Discrete Register
W            | Word Register
R            | Real Register
DX           | Discrete Input
DY           | Discrete Output
WX           | Word Input
WY           | Word Output
